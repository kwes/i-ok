# i-ok

## short desc
small math esolang for small numbers and equations

## how to use
| chars   | desc |
| --------| --------|
| i+ or i++| adds    |
| i- or i--| subtracts|
| i*      | multiples by 2|
| i/      | divides by 2|

```
i+ i+ i- i* this would equal 2
```
### letters
going over 15 turns numbers into letters
```
i+ i+ i* i* i* this would be 'a'
```
max letters is a-i
going over 24 turns result into null
